# Product Overview

SpaceCoding Presentation Agent is a multi-format content creation platform designed for WeChat content creators. It transforms written articles into multiple distribution formats through an AI-powered workflow.

## Core Concept

Central Kitchen approach - one source material (WeChat article) generates multiple content assets:
- Short videos with AI-generated visuals and narration
- Presentation slides (PPT/web slides)
- Social media posters (Xiaohongshu/Instagram format)

## Workflow Stages

1. **Research** - Topic ideation and research using AI
2. **Story** - Article writing with intelligent CMS editor
3. **Poster** - Extract golden quotes for social media cards
4. **Script** - A2S (Article-to-Storyboard) engine breaks article into video scenes
5. **Visual** - AI generates HTML slides with animations
6. **Export** - Final video synthesis and timeline editing

## Key Features

- AI-powered content generation using Google Gemini
- Real-time TTS (text-to-speech) with timeline markers
- Tiptap rich text editor for article writing
- GitHub integration for content management
- Multi-stage project dashboard
- Responsive 16:9 slide rendering
