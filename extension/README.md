# AI 全媒体发布助手 - Chrome 插件

AI 驱动的跨平台内容发布工具，自动识别页面元素并填充内容。

## 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `extension` 文件夹

## 使用方法

1. 在 React 应用中写好文章
2. 进入「发布」阶段
3. 点击目标平台（微信/知乎/小红书/头条）
4. 插件自动打开平台页面并填充内容
5. 检查内容后手动点击发布

## 工作原理

1. **DOM 快照提取** - 扫描页面交互元素
2. **AI 分析** - Gemini 分析页面结构，返回操作步骤
3. **自动执行** - 模拟人类操作填充内容

## 支持平台

- 微信公众号 (mp.weixin.qq.com)
- 知乎专栏 (zhuanlan.zhihu.com)
- 小红书 (creator.xiaohongshu.com)

## 注意事项

- 需要先登录各平台账号
- 图片可能需要手动处理
- 建议发布前检查内容格式
